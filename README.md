# SimPassengers — Public Downloads

This repository hosts the public release artifacts (Windows installers, checksums, release notes) for **SimPassengers**, the virtual passenger experience for **Microsoft Flight Simulator 2024**.

The source code is proprietary and not included here. This repository exists solely to deliver downloadable installers from a stable URL pattern with version tagging and CDN-backed delivery.

## What you need to run SimPassengers

- **Microsoft Flight Simulator 2024** (Store or Steam edition)
- **Windows 10** or **Windows 11**
- A **SimPassengers account** — create one at [simpassengers.com/register](https://simpassengers.com/register)
- An **active licence** — purchase from your dashboard at [simpassengers.com/dashboard](https://simpassengers.com/dashboard) (Basic $24.99 lifetime or Supporter $49.99 lifetime)

---

## Installation

1. **Close MSFS 2024** if it's running.
2. **Download** the installer from the [latest release](https://github.com/anilacesolutions/simpassengers-releases/releases/latest).
3. **Run** the installer. Windows SmartScreen may show an _Unknown publisher_ warning during the early launch window — click **More info → Run anyway**. The installer is unsigned for v1; code signing is on the post-launch roadmap.
4. **Launch SimPassengers** from the Start menu or desktop shortcut, **sign in** with your account, then **start MSFS 2024**. The bridge will auto-start with the simulator from this point on.
